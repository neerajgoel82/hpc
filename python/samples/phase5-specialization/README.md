Choose your path! Pick one or more specializations based on your interests and career goals.

## Available Paths

- Flask or Django framework
- HTML/CSS basics
- REST APIs
- Database integration (SQLite, PostgreSQL)
- Authentication and sessions
- Deployment basics

**Projects:**
- Personal blog
- Todo app with API
- URL shortener
- Weather dashboard

- `web/01_flask_basics.py`

## How to Choose

**Web Development**: Want to build websites and web apps
**Data Science**: Interested in data analysis and visualization
**Automation**: Want to save time with scripts
**CLI Tools**: Love the command line and developer tools
**API Development**: Want to build backends for apps

## Multi-Path Approach

You can absolutely pursue multiple paths! Many concepts overlap:
- Web Dev + API Development = Full stack
- Data Science + Automation = Data pipelines
- CLI + Automation = Powerful utilities

## Getting Started

1. Choose your path(s)
2. Navigate to the appropriate directory
3. Start with lesson 01
4. Complete the capstone project

```bash
# Example: Starting Web Development path
cd phase5-specialization/web
python3 01_flask_basics.py
```
