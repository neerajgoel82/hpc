# Phase 2: Intermediate Python (Weeks 4-6)

You've mastered the basics! Now let's build on that foundation with functions, file operations, and modules.

## What You'll Learn

- Functions and parameters
- Error handling
- File I/O operations
- Working with CSV and JSON
- Modules and packages
- Virtual environments

## Week-by-Week Breakdown

### Week 4: Functions
- `16_functions_basics.py` - Defining and calling functions
- `17_function_parameters.py` - Parameters, arguments, defaults
- `18_return_values.py` - Return statements and multiple returns
- `19_scope.py` - Variable scope and namespaces
- `20_lambda_functions.py` - Lambda functions and functional programming
- `21_recursion.py` - Recursive functions
- **Project:** `project_text_analyzer.py` - Text analysis tool

### Week 5: Error Handling & Files
- `22_exceptions.py` - Try-except blocks
- `23_file_reading.py` - Reading files
- `24_file_writing.py` - Writing files
- `25_csv_files.py` - Working with CSV
- `26_json_files.py` - Working with JSON
- **Project:** `project_file_organizer.py` - Automated file organizer

### Week 6: Modules & Packages
- `27_importing.py` - Import statements
- `28_standard_library.py` - Common standard library modules
- `29_creating_modules.py` - Writing your own modules
- `30_packages.py` - Creating and using packages
- `31_virtual_environments.py` - Managing dependencies
- **Project:** `project_cli_tool.py` - Command-line utility

## Getting Started

```bash
cd phase2-intermediate
python3 16_functions_basics.py
```

## Prerequisites

Make sure you've completed Phase 1 and are comfortable with:
- Variables, data types, operators
- Control flow (if/else, loops)
- Data structures (lists, dicts, sets, tuples)
- String manipulation

Ready to level up? Let's go!
